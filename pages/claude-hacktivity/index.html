Setup Claude Code

You should have received an invite to Claude Console. Follow these steps to get set up:

  1. Accept the invite sent to your Dartmouth email
  2. Log into Claude Console at console.anthropic.com
  3. Install Claude Code CLI:

Tips for Using Claude Code

  • Set your style to "explainer" - Run /config and set the style to explainer. This helps Claude explain what it's doing as it works.
  • Use plan mode heavily - Before building, ask Claude to plan first. This helps you understand the approach before code gets written. You can enter plan mode by typing /plan or asking Claude to "plan this out first".
  • Use CLAUDE.md for memory - If you need Claude to remember something for future sessions, ask it to add it to CLAUDE.md. This is a special file that Claude reads at the start of every conversation â€” it's like giving Claude persistent notes about your project, preferences, or conventions.

The Activity

Step 1: Discuss (5-10 min)

With your partner, discuss your need finding interviews:

  • What problems did you uncover?
  • What patterns or insights stood out?
  • What surprised you?

Step 2: Brainstorm with Claude (10-15 min)

Open Claude Code in your cloned repo and start a conversation:

  • Share your need finding insights with Claude
  • Ask it to help brainstorm potential solutions
  • Explore different approaches together
  • Narrow down to one idea to prototype

Step 3: Build a Quick Prototype (30-40 min)

Use Claude Code to build a frontend-only prototype:

  • No backend, no database - just UI
  • Claude can test it in the browser as you go (via Chrome integration)
  • Focus on the core user experience
  • It doesn't need to be perfect - this is a quick exploration
